# AGENTS Instructions

## Commit Messages

- When Codex proposes a commit message for this repository, it must use the Conventional Commits format.
- Preferred pattern: `type(scope): summary` when a useful scope exists.
- Fallback pattern: `type: summary` when no clear scope exists.
- Keep the summary concise and imperative.

## Examples

- `feat(pages): add landing page links`
- `fix(editor): handle Pyodide load errors`
- `refactor: move Python editor to dedicated page`
